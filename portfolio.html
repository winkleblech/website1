1.2.5 Project: Burger Collect

We created a program using python allowing you to collect burger elements on a bun.

You get to choose the order and which elements you want in your burger including tomatos,

cheese, onions, lettuce, and patties. There is a score to see how many the user can collect.

At the end a leaderboard is shown and the user gets to see if they had a high enough score.


First Scratch Project: Starfish Pong Game

We created a scratch game using starfish sprites, an ocean background, a score, and

displaying the highest score. Two starfish are boucing across the screen and the user

has to use a green horizontal pong paddle to bounce the starfish away from the red line

on the bottom. Once one starfish hits the bottom, the game terminates high score is recorded.


1.3.6 Rover Phone Home

We believe that the rover is located in the Rocky Mountains terrain. We reached this conclusion

by creating predicted graphs for each of the 4 elements for each of the 5 terrains. From there,

we created graphs from the actual data collected. By comparing the actual and predicted graphs,

we were able to notice similarites and determine which graph represented wind, sound, light, and

temperature. This further allowed us to realize how the rocky mountain terrain graphs matched up

the best out of the 4 other terrains.
